📚 Lista de Exercícios - Formação Front-End

Pratique seus conhecimentos em HTML, CSS e JavaScript

🏗️ SEÇÃO 1: HTML & CSS Básico

Exercício 1.1 - Página de Portfólio Pessoal

Dificuldade: ⭐ Fácil

Crie uma página de portfólio com sua informação pessoal incluindo seções: Sobre Mim, Projetos e Contato.

💡 Dicas:
  • Use HTML semântico: <header>, <main>, <section>, <footer>
  • Estruture com divs e classes bem nomeadas para o CSS
  • Adicione uma paleta de cores consistente
  • Use flexbox para alinhar elementos horizontalmente

Exercício 1.2 - Galeria de Imagens Responsiva

Dificuldade: ⭐⭐ Médio

Crie uma galeria de imagens que se adapta a diferentes tamanhos de tela (desktop, tablet, mobile).

💡 Dicas:
  • Use @media queries para diferentes breakpoints
  • Implemente com CSS Grid para layout responsivo
  • Adicione meta tag viewport: <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • Use unidades relativas: %, em, rem ao invés de px
  • Teste em diferentes resoluções (mobile: 320px, tablet: 768px, desktop: 1024px+)

Exercício 1.3 - Formulário de Cadastro com Validação CSS

Dificuldade: ⭐⭐ Médio

Crie um formulário completo com campos de texto, email, senha, select e checkbox com estilos visuais.

💡 Dicas:
  • Use seletores CSS: :focus, :hover, :valid, :invalid
  • Implemente labels associadas com atributo for
  • Use <fieldset> e <legend> para agrupar campos
  • Adicione feedback visual com borders e cores para inputs válidos/inválidos
  • Estruture com flexbox para alinhar campos verticalmente

Exercício 1.4 - Clone de Landing Page

Dificuldade: ⭐⭐⭐ Desafiador

Recrie uma landing page de uma empresa/produto real (SaaS, e-commerce, etc) focando em layout e responsividade.

💡 Dicas:
  • Analise a página original e identifique as seções principais
  • Use CSS Grid para layouts mais complexos
  • Implemente um menu de navegação responsivo
  • Adicione efeitos com :hover em botões e links
  • Use imagens de placeholder (Unsplash, Pexels)

⚙️ SEÇÃO 2: JavaScript Básico

Exercício 2.1 - Conversor de Temperatura

Dificuldade: ⭐ Fácil

Crie uma calculadora que converte temperaturas entre Celsius, Fahrenheit e Kelvin.

💡 Dicas:
  • Use const e let para declarar variáveis
  • Implemente funções para cada conversão: celsiusParaFahrenheit()
  • Use operadores aritméticos: +, -, *, /
  • Crie um input HTML para o usuário inserir a temperatura
  • Valide se o input é um número válido com isNaN() ou typeof

Exercício 2.2 - Calculadora IMC (Índice de Massa Corporal)

Dificuldade: ⭐⭐ Médio

Crie uma calculadora de IMC que retorna a categoria de peso e um feedback personalizado.

💡 Dicas:
  • Fórmula: IMC = peso / (altura²)
  • Use estruturas condicionais: if/else ou switch
  • Categorias: Abaixo do peso (<18.5), Normal (18.5-24.9), Sobrepeso (25-29.9), Obeso (>30)
  • Use document.getElementById() ou document.querySelector() para acessar elementos
  • Adicione o resultado dinamicamente com innerHTML

Exercício 2.3 - Validador de CPF

Dificuldade: ⭐⭐ Médio

Crie um validador de CPF que verifica o formato e calcula os dígitos verificadores.

💡 Dicas:
  • Use string.length para verificar o tamanho
  • Use string.split('') para dividir em caracteres
  • Use loops for para iterar e validar os dígitos
  • Implemente a fórmula do primeiro dígito verificador (soma ponderada)
  • Repita para o segundo dígito verificador

Exercício 2.4 - Jogo da Velha (Tic Tac Toe)

Dificuldade: ⭐⭐⭐ Desafiador

Implemente o jogo da velha com lógica de verificação de vitória e alternância de jogadores.

💡 Dicas:
  • Use um array 3x3 para representar o tabuleiro
  • Implemente função para verificar vitória: verificar linhas, colunas e diagonais
  • Use addEventListener para capturar cliques nos botões
  • Controle turnos com variável booleana ou contador
  • Adicione função para resetar o jogo

📦 SEÇÃO 3: Arrays e Objetos

Exercício 3.1 - Lista de Tarefas (TODO List)

Dificuldade: ⭐⭐ Médio

Crie um app de lista de tarefas com funcionalidades de adicionar, remover e marcar como completo.

💡 Dicas:
  • Use array de objetos: [{id: 1, texto: "Tarefa", completo: false}]
  • Implemente métodos: push() para adicionar, splice() para remover
  • Use filter() para remover tarefas completas
  • Use map() para renderizar lista no DOM
  • Salve dados no localStorage para persistência

Exercício 3.2 - Catálogo de Produtos

Dificuldade: ⭐⭐ Médio

Crie um catálogo com produtos (nome, preço, categoria) com funcionalidades de filtro e busca.

💡 Dicas:
  • Estruture produtos como objetos: {id, nome, preco, categoria, descrição}
  • Use filter() para filtrar por categoria
  • Use find() ou filter() com includes() para buscar por nome
  • Use sort() para ordenar por preço
  • Renderize cards de produtos dinamicamente com innerHTML ou createElement()

Exercício 3.3 - Calculadora de Estatísticas

Dificuldade: ⭐⭐⭐ Desafiador

Crie uma calculadora que retorna média, mediana, moda e desvio padrão de um conjunto de números.

💡 Dicas:
  • Média: Use reduce() para somar e dividir por length
  • Mediana: sort() o array e pegue o elemento do meio
  • Moda: Use reduce() para contar frequências em um objeto
  • Desvio Padrão: Use fórmula matemática com map() e Math.sqrt()
  • Separe em funções: calcularMedia(), calcularMediana(), etc.

Exercício 3.4 - Gerenciador de Contatos

Dificuldade: ⭐⭐⭐ Desafiador

Crie um app que gerencia contatos com opções de adicionar, editar, remover e buscar.

💡 Dicas:
  • Estruture contatos: {id, nome, email, telefone, categoria}
  • Implemente funções: adicionarContato(), editarContato(), removerContato()
  • Use find() para buscar por ID ou nome
  • Valide email com regex: /^[^\s@]+@[^\s@]+\.[^\s@]+$/
  • Persista dados no localStorage

🏛️ SEÇÃO 4: Programação Orientada a Objetos

Exercício 4.1 - Sistema de Banco

Dificuldade: ⭐⭐ Médio

Crie classes para representar Banco, Conta e Cliente com operações de depósito, saque e transferência.

💡 Dicas:
  • Crie classe Cliente com propriedades: nome, CPF
  • Crie classe Conta com: titular, saldo, numero, agencia
  • Implemente métodos: depositar(), sacar(), consultarSaldo()
  • Use constructor() para inicializar propriedades
  • Adicione validações: saldo insuficiente, valores negativos

Exercício 4.2 - Sistema de Biblioteca

Dificuldade: ⭐⭐⭐ Desafiador

Crie classes para Livro, Autor, Biblioteca e Usuario com empréstimos e devoluções.

💡 Dicas:
  • Classe Livro: title, author, ISBN, disponivel
  • Classe Usuario: nome, ID, livrosEmprestados (array)
  • Classe Biblioteca: acervo (array de livros), usuarios
  • Métodos: emprestarLivro(), devolverLivro(), listarDisponíveis()
  • Use herança se necessário para tipos de usuários diferentes

Exercício 4.3 - Jogo de RPG com Classes

Dificuldade: ⭐⭐⭐ Desafiador

Crie um mini jogo RPG com classes de Personagem, Guerreiro, Mago com sistemas de combate.

💡 Dicas:
  • Classe base Personagem: nome, vida, mana, nivel, experiencia
  • Subclasses: Guerreiro (força), Mago (magia)
  • Implemente herança com extends e super()
  • Métodos de combate: atacar(), defender(), usarMagia()
  • Sistema de experiência e level up

Exercício 4.4 - Sistema de E-commerce

Dificuldade: ⭐⭐⭐ Desafiador

Crie um sistema completo com Produto, Carrinho, Pedido e Cliente com cálculo de impostos.

💡 Dicas:
  • Classe Produto: id, nome, preco, estoque, categoria
  • Classe ItemCarrinho: produto, quantidade
  • Classe Carrinho: itens[], metodos: addItem, removeItem, calcularTotal
  • Classe Pedido: numero, cliente, carrinho, data, status
  • Implemente cálculo de desconto, imposto e frete

⚡ SEÇÃO 5: Programação Assíncrona

Exercício 5.1 - Requisição de API com Fetch

Dificuldade: ⭐⭐ Médio

Crie uma página que consome uma API pública (JSONPlaceholder, OpenWeather) e exibe os dados.

💡 Dicas:
  • Use fetch(url) para fazer requisição GET
  • Use async/await para melhor legibilidade
  • Trate erros com try/catch
  • Use .json() para converter resposta
  • Exiba um loading enquanto a requisição está sendo feita

Exercício 5.2 - Buscador de Usuários GitHub

Dificuldade: ⭐⭐ Médio

Crie um app que busca usuários do GitHub e exibe seus repositórios e informações.

💡 Dicas:
  • Use API do GitHub: https://api.github.com/users/{username}
  • Busque repositórios: https://api.github.com/users/{username}/repos
  • Implemente debounce para não fazer requisição a cada caractere digitado
  • Adicione validação se o usuário existe
  • Exiba avatar, bio, localização e repositórios com links

Exercício 5.3 - Previsão do Tempo

Dificuldade: ⭐⭐⭐ Desafiador

Crie um app de previsão do tempo usando uma API (OpenWeatherMap) com dados atualizados.

💡 Dicas:
  • Use API OpenWeatherMap (com chave gratuita)
  • Obtenha localização do usuário com navigator.geolocation
  • Ou permita busca por cidade
  • Exiba temperatura atual, sensação térmica, umidade, velocidade do vento
  • Mostre previsão para próximos dias com ícones apropriados

Exercício 5.4 - Sistema de Chat com WebSocket

Dificuldade: ⭐⭐⭐ Desafiador

Crie um chat em tempo real usando WebSocket (socket.io ou nativo WebSocket).

💡 Dicas:
  • Use socket.io para conexão em tempo real (mais fácil)
  • Ou use WebSocket nativo: new WebSocket('ws://...')
  • Implemente evento de conexão e desconexão
  • Envie/receba mensagens com eventos
  • Exiba lista de usuários conectados
  • Adicione timestamps nas mensagens

🎯 SEÇÃO 6: Projetos Integrados (HTML + CSS + JS)

Exercício 6.1 - Aplicativo de Notas

Dificuldade: ⭐⭐⭐ Desafiador

Crie um app completo de notas com CRUD (criar, ler, atualizar, deletar), categorias e busca.

💡 Dicas:
  • Use localStorage ou IndexedDB para persistência
  • Implemente interface responsiva (desktop e mobile)
  • Adicione categorias com cores diferentes
  • Funcionalidades: criar, editar, deletar, arquivar notas
  • Busca por texto ou categoria
  • Ordenação por data de criação/modificação

Exercício 6.2 - Pomodoro Timer

Dificuldade: ⭐⭐ Médio

Crie um timer Pomodoro com trabalho, pausa curta e pausa longa.

💡 Dicas:
  • Use setInterval() e clearInterval() para controlar o timer
  • Padrão Pomodoro: 25 min trabalho, 5 min pausa curta, 15 min pausa longa
  • Botões: iniciar, pausar, reiniciar
  • Exiba o tempo em formato MM:SS
  • Adicione notificação sonora quando termina
  • Interface visual que muda de cor em cada fase

Exercício 6.3 - Gerenciador de Despesas

Dificuldade: ⭐⭐⭐ Desafiador

Crie um app de controle de despesas com categorias, gráficos e relatórios.

💡 Dicas:
  • Categorias: alimentação, transporte, saúde, lazer, outros
  • Campos: descrição, valor, categoria, data
  • Funcionalidades: adicionar, editar, deletar, filtrar por categoria/período
  • Use Chart.js ou Recharts para gráficos
  • Calcule total gasto, por categoria, por período
  • Exporte dados em CSV ou PDF

Exercício 6.4 - Blog com Comentários

Dificuldade: ⭐⭐⭐ Desafiador

Crie um blog com posts, comentários, likes e sistema de tags.

💡 Dicas:
  • Estrutura de dados: posts com título, conteúdo, autor, data, tags
  • Comentários: autor, texto, data, likes
  • Sistema de tags para filtrar posts
  • Busca por título ou conteúdo
  • Editor markdown ou rich text para criar posts

Exercício 6.5 - Rede Social Simples

Dificuldade: ⭐⭐⭐⭐ Muito Desafiador

Crie uma rede social com usuários, posts, comentários, likes, seguidores e mensagens.

💡 Dicas:
  • Estrutura: Usuários, Posts, Comentários, Mensagens, Seguidores
  • Funcionalidades: login/registro, perfil, criar post, comentar, like
  • Seguidores: seguir/deseguir usuários, feed personalizado

  • Notificações de likes e comentários
