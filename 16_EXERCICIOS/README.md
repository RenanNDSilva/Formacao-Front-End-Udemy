## 🎯 Como Usar

### 1. **Abra o arquivo `index.html`**
   - Abra a página no navegador para ver todos os exercícios organizados
   - Cada exercício tem dificuldade marcada e dicas específicas

### 2. **Escolha um exercício**
   - Comece pelos exercícios ⭐ (Fácil)
   - Avance progressivamente para ⭐⭐ (Médio), ⭐⭐⭐ (Desafiador), etc.

### 3. **Siga as dicas**
   - Cada exercício inclui funções/estruturas a usar
   - Use os links de referência fornecidos

### 4. **Crie sua solução**
   - Crie uma pasta para cada exercício dentro de `16_EXERCICIOS/`
   - Exemplo: `16_EXERCICIOS/exercicio_2_1_conversor_temperatura/`
   - Organize com `index.html`, `css/styles.css`, `js/scripts.js`

### 5. **Valide seu código**
   - Teste no navegador
   - Verifique se funciona em mobile também
   - Compare sua solução com possíveis alternativas

### Ferramentas Úteis

1. **DevTools do Navegador**
   - F12 ou Ctrl+Shift+I para abrir
   - Use para debugar e testar código

2. **Validadores Online**
   - HTML: https://validator.w3.org/
   - CSS: https://jigsaw.w3.org/css-validator/
   - JavaScript: ESLint online

3. **APIs Públicas Úteis**
   - JSONPlaceholder: https://jsonplaceholder.typicode.com/
   - OpenWeatherMap: https://openweathermap.org/api
   - GitHub API: https://api.github.com/

4. **Bibliotecas JavaScript Úteis**
   - Chart.js - Gráficos
   - Prism.js - Syntax Highlight
   - Socket.io - WebSocket
   - Moment.js - Manipulação de datas
